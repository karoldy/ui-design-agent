这是一篇关于 AI Agent 的技术架构设计笔记，涵盖 多智能体协作、记忆管理与推理规划等核心模块。

系统架构概览

整体架构分为三个层次：感知层、推理层和执行层。感知层负责多模态输入解析，推理层基于 LLM 进行决策规划，执行层调用外部工具完成具体任务。

一个 Agent 的核心能力取决于其记忆系统的设计 —— 长期记忆决定了 Agent 能否从历史经验中学习，短期记忆则影响对话上下文的理解连贯性。

关键组件

Tool Use：通过 Function Calling 调用外部 API，实现信息检索、代码执行等操作
Memory Manager：管理短期对话缓存与长期向量存储，支持 RAG 检索增强
Planner：基于 ReAct 模式的思考-行动-观察循环，动态调整执行策略
