空状态组件 — Empty State

各种空状态场景的复用组件。采用居中布局、极简 SVG 插画（或 MDI 图标）风格。 统一使用 accent 色的极简线条风格，所有空状态保持一致的视觉语言。

设计规范 Design Specs

插画风格
  • 极简线性 + 单色（accent blue）
  • 线条粗细：1.5-2px
  • 圆角端点，保持干净感
  • 图标替代方案：MDI icon 72px 圆形容器
排版规范
  • 主标题：18px, 600 weight
  • 副标题：13px, secondary 色, max 320px 宽
  • CTA 按钮：accent 色 solid
  • 居中对齐，不打扰周围 UI
