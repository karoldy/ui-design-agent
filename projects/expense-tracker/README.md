# Daily Expense Tracker (个人记账工具)

极简记账微信小程序 — **打开即记、智能分类、零广告**。

面向「想记账但坚持不下来」的年轻上班族，通过最大化降低记录摩擦来培养记账习惯。核心差异：首屏数字键盘聚焦、自动分类推断、完全无广告、100% 离线可用。

---

## 设计目标

| 目标 | 说明 |
|------|------|
| **10秒完成一笔记账** | 打开小程序 → 输入金额 → 确认保存，三步完成 |
| **零学习成本** | 首次用户无需任何引导即可完成第一笔记账 |
| **账面感知清晰** | 3秒内回答「这个月花了多少、花在哪、趋势如何」 |
| **分类不纠结** | 基于时段+金额的智能分类规则，推荐正确率 > 70% |

## 约束条件

| 约束 | 说明 |
|------|------|
| 平台 | 微信小程序（零安装，即开即用） |
| 存储 | 本地存储优先，不强制注册登录，不上传数据 |
| 性能 | 冷启动 < 1.5s，保存响应 < 200ms |
| 品牌 | 完全无广告，无社交分享，无签到/积分系统 |
| 合规 | 不获取用户手机号、位置等隐私信息 |

## 关键决策

| 日期 | 决策 | 理由 |
|------|------|------|
| 2026-07-02 | 首页默认不显示分类选择器，用规则推断 + 允许手动修正 | 用户最核心的放弃原因是指标太多；自动推断可减少 70% 以上的分类操作次数 |
| 2026-07-02 | 底栏 3 个 Tab：记账 / 统计 / 记录 | 2 Tab 在信息密度上不够，3 Tab 能清晰分离「录入」「分析」「回顾」三个心智模型 |
| 2026-07-02 | 自定义数字键盘替换系统键盘 | 确保全机型表现一致，可集成确认键缩短操作路径，避免系统键盘切换输入法的干扰 |
| 2026-07-02 | 使用 Emoji 作为分类图标 | 零资源加载、无版权问题、用户辨识度高、不增加包体积 |
| 2026-07-02 | 不配置分享功能，不监听 onShareAppMessage | 符合「安静的工具」定位，不做病毒传播，用户更信任 |
| 2026-07-02 | 图表使用 Canvas 2D 原生绘制，不引入第三方图表库 | 控制包体积在 50KB 以内，保证冷启动速度 |
| 2026-07-02 | 6:00-9:30 时段 + 5-30元 = 餐饮(早餐)，以此类推 | 基于典型用户生活习惯的简单规则引擎，不需要训练数据的轻量方案 |

## Tab 结构

```
Tab 1: 首页（记账）          默认页。数字键盘 + 金额输入 + 分类标签 + 语音入口
Tab 2: 统计                  饼图 + 分类排行 Top5 + 按周/月趋势折线图
Tab 3: 记录                  按日期分组的支出时间线，左滑删除，点击编辑
```
